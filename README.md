# TermTrainer

macOS 终端命令练习器 — 通过交互式练习快速掌握常用终端命令。

## 功能特性

- **10 大命令分类**，覆盖 64 个常用命令、106 道练习题
- **分类练习** — 按模块逐个击破
- **随机练习** — 随机抽取 10 题综合测试
- **错题本** — 自动记录错题，针对弱点反复巩固
- **进度追踪** — 正确率、连胜记录、分类统计一目了然
- **成就系统** — 6 枚徽章，激励持续练习
- **零依赖** — 纯 Python 3 标准库，开箱即用

## 快速开始

### 环境要求

- Python 3.6+
- macOS / Linux 终端

### 运行

```bash
# 方式一：直接运行
python3 termtrainer.py

# 方式二：使用启动脚本
chmod +x start.sh
./start.sh
```

## 使用说明

### 主菜单

```
[1-10]  选择命令分类进行练习
[a]     查看所有命令参考手册
[r]     随机练习（10题）
[w]     错题本
[s]     查看详细统计
[h]     帮助
[q]     退出
```

### 练习中的快捷指令

| 指令 | 说明 |
|------|------|
| `hint` | 获取提示（显示命令开头和示例） |
| `skip` | 跳过当前题目 |
| `quit` | 退出当前练习，返回上级菜单 |

### 错题本

答错或跳过的题目会自动记录到错题本。在主菜单按 `[w]` 进入错题本：

- 查看所有错题及错误次数
- 开始错题专项练习 — 答对的题目自动从错题本移除
- 支持一键清空错题本

错题按错误次数排序，帮助你优先攻克最薄弱的知识点。

### 成就系统

| 成就 | 条件 |
|------|------|
| 初学者 | 完成 10 道练习 |
| 练习达人 | 完成 50 道练习 |
| 终端大师 | 完成 100 道练习 |
| 连胜新秀 | 连续答对 5 题 |
| 连胜达人 | 连续答对 10 题 |
| 连胜大师 | 最佳连胜达到 20 题 |

## 数据存储

练习进度保存在 `~/.termtrainer/progress.json`，包括：

- 总练习数、正确数、正确率
- 连胜记录
- 分类和命令维度的统计
- 错题本数据
- 已解锁的成就
